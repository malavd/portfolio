# How to Add Blog Link to Portfolio

When ready to publish, add this to your portfolio navigation in index.html:

Find the navigation section (around line 46) and add:
<a href="blog/" class="nav-link">Blog</a>

Example:
<nav class="site-nav" id="site-nav">
  <a href="#intro" class="nav-link">Intro</a>
  <a href="#what-i-do" class="nav-link">What I Do</a>
  <a href="#experience" class="nav-link">Experience</a>
  <a href="#skills" class="nav-link">Skills</a>
  <a href="#education" class="nav-link">Education</a>
  <a href="#certification" class="nav-link">Certification</a>
  <a href="blog/" class="nav-link">Blog</a>
  <a href="#contact" class="nav-link">Contact</a>
</nav>
