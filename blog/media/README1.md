# Blog Media Files

## Directory Structure

Store images and videos organized by date:
- images/2026-02/ (for February 2026 posts)
- videos/2026-02/ (for February 2026 videos)

## Guidelines

### Images
- Use JPG for photos, PNG for diagrams
- Keep under 2MB per image
- Name files descriptively (e.g., "tara-process-diagram.png")

### Videos
- Use MP4 format
- Keep under 50MB per video
- Create thumbnails for video previews
